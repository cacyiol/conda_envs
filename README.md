# conda_envs
some conda envs for open source projects
在配置开源项目的环境时，有时候明明是按照environment.yaml或者requirements.txt安装的，在启动时还是会出现层出不穷的问题，因此笔者将自己曾使用过的开源项目的环境上传到github，下载后放到anaconda的envs目录下即可使用。
